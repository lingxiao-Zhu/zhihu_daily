# 知乎日报_iOS

## 介绍
+ 基于知乎日报API进行开发，仅用于学习交流；
  + 语言：Objective-C;
  + 基于MVC模式；
  + 布局使用Masonry；
  + 网络请求使用AFNetworking；
  + sdwebimage进行图片异步加载；
  + 依赖管理：cocoapods
